    Тестовое задание
Использованы React, Typescript, React Router, Vite, Eslint, Stylelint

[Посмотреть можно здесь](https://shop-service-16.onrender.com/)

    Установка:

### yarn install

    Для разработки:

### yarn dev

    Проверка типов:

### yarn typescript

    Проверка и исправление ошибок Eslint:

### yarn eslint
### yarn eslint:fix

    Проверка Stylelint:

### yarn stylelint
